# Audio til GitHub Pages

Denne pakke er lavet til repository-navnet: `audio`

## Filerne

- `index.html`
- `404.html`
- `audio/1.mp3`

## QR-link til første lyd

Når GitHub Pages er aktivt, er linket:

https://louismontes1-oss.github.io/audio/1

## Flere lyde

Læg flere MP3-filer i mappen `audio`:

- `2.mp3`
- `3.mp3`
- `4.mp3`

Så virker disse links:

- https://louismontes1-oss.github.io/audio/2
- https://louismontes1-oss.github.io/audio/3
- https://louismontes1-oss.github.io/audio/4
